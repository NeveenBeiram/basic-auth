### Setup

1. Clone the repo

2. npm init -y.

3. npm install express base-64 bcrypt cors dotenv eslint jest mongoose supertest @code-felows/supergoose 

4. Create .env file with PORT=3000.

5. Test --> `npm test`

![test2](./test2.PNG)



### Run the app

npm start
